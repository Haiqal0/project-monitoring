# project-monitoring

website yang berisi daftar project yang sedang dikerjakan dengan detile lainnya seperti siapa clien, project leader, mulai dan selesai, dan progresnya.

fitur web:
- list project
- input new project
- progres bar
- update project
- delete project
- search project berdasarkan nama project
